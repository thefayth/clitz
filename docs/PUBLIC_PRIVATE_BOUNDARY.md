# Public / Private Boundary

The CLITZ public surface is designed to share enough for legitimacy, collaboration, and launch visibility while protecting the operational system.

## Public

- Project story and mission
- Approved images and diagrams
- Status and roadmap
- WordPress page draft
- Public interface names
- Launch checklist
- Privacy and safety posture
- Ownership and rights notices

## Private

- WordPress plugin source
- Desktop app source
- Prompts and private workflows
- Raw moderation queues
- Credentials and deployment details
- Server configuration
- Vault and backup material
- Sensitive records
- Admin screenshots
- Adult/private creator operations
- Customer, family, medical, legal, benefits, or private personal data

## Rule

Public materials may describe capability, interface, and governance.

Private materials hold implementation, operations, raw evidence, credentials, queues, and records.
